# Hades 2 Rogue Archipelago

Archipelago multiworld integration for Hades 2 Rogue. Mod was created with AI assistance (Tag is missing for Hades 2, will add the tag when it becomes available)



This mod connects to the Archipelago Hades 2 Rogue client. MAKE SURE you connect to the server before launching into a NEW save.



## Notes

* Ensure you've connected to the server via the Hades 2 Rogue client before you launch into a save.
* Make sure that the save is a new save
* If you set the starting area to surface, it will kill you at the beginning of the intro run, since it takes place in the underworld. I haven't found a way around this yet.
* This is not completely stable yet and expect to run into bugs. Feel free to inform me of those bugs on the Hades 2 Rogue Archipelago discord: https://discord.com/channels/731205301247803413/1521929337954238624
