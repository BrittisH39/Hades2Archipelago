# Hades II Archipelago

Archipelago multiworld integration for Hades II.

This mod connects to the Archipelago Hades 2 client. MAKE SURE you connect to the server before launching into a NEW save.



## Notes

* Ensure you've connected to the server via the Hades 2 client before you launch into a save.
* Make sure that the save is a new save
* If you set the starting area to surface, it will kill you at the beginning of the intro run, since it takes place in the underworld. I haven't found a way around this yet.
* This is not completely stable yet and expect to run into bugs. Feel free to inform me of those bugs on the Archipelago discord: https://discord.com/channels/731205301247803413/1375345291926044722
