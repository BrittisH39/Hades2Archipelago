# Hades 2 Archipelago

Hades 2 implementation for the Archipelago Multiworld Randomizer - Created by BrittisH39 with heavy AI assistance.



Windows only at this time



## **Setup**

#### 

#### **Mod**

* Install [r2modman](https://thunderstore.io/c/riskofrain2/p/ebkr/r2modman/)
* Launch, then find and select Hades 2. Select which store you own Hades 2 on (Epic Games Launcher untested at this time)
* Select or create a profile.
* Select online on the left side, and search and find "Hades2RogueArchipelago"
* Select download, then Download with Dependencies



#### **APWorld/Hades 2 Client**

* Download the APWorld from the releases tab
* Download and install [Archipelago](https://archipelago.gg/tutorial/Archipelago/setup_en) if you haven't already



**Then EITHER**

* Open the Archipelago Launcher
* Click and drag the Hades\_2.apworld file onto the Archipelago Window
* It will give you a confirmation message, relaunch Archipelago, and you're done.

**OR**

* Open you Archipelago installation
* Find the "Custom\_worlds" folder
* Place the Hades2Rogue.apworld file into that folder



* Search for the Hades 2 Client in the archipelago launcher
* Launch, connect to a server, select your slot.
* Launch the game with mods from r2modman
* Start a NEW SAVE
* You should be good to go! If you set up to start with Surface access, you'll be instantly killed and will start from the crossroads, else everything should work from the intro run.



# **Notes**



This mod is not fully stable, you should expect bugs. I have been able to finish runs, but you will inevitably run into issues. Please feel free to report those on the [Discord](https://discord.com/channels/731205301247803413/1375345291926044722)

If you start with a Surface run, it will instantly kill you. That's normal it shouldn't send a deathlink

Currently the multiplying locations feature isn't working, so if you only choose one route using Room\_based, there will be unreachable locations. So.. maybe don't do that.

Some of the "Starting \_\_\_\_" items and other filler checks aren't quite working how I want them to yet.

To move/resize the mod overlay, it should be Insert on your keyboard
